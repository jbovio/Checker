Graphic interface
=======================

Buttons
---------

Different buttons that we can find:

:Check: Execute check function, depending on the result can enable more buttons.
:Fix: Fix the issue, or flag the related objects. *(usually)*
:Sel: Select the objects related to the checker.
:Check all: Execute all check functions registered in the project:step list.
