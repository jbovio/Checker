Set new project

Setps

  1. open ../lib/projectlist.py

2. Inside projectlist.py there is a function called “projectFilters” in order to set a new project required add a new key in projectFilter dictionary and indicate the check functions per step and project. 3. e.g. {projectName: { step: check function }, { step2: check funtion }} 4. Some funtions required specific parameters can be set in the key value.
